Released under the GNU license, this is a free Joomla extension which enables you to obfuscate email addresses to protect them from harvesting by internet bots.

If you will need to know more, go to: https://www.unisoftdev.tech/download/mask-email

This plugin on the official Joomla directory with a long description: https://extensions.joomla.org/profile/profile/details/325855/
